Dicas para Resolver Exercícios

Proposições

Lembre-se que uma proposição é uma sentença declarativa que pode ser verdadeira ou falsa, mas não ambas.

Condicional

A proposição "Se p então q" só é falsa quando p é verdadeiro e q é falso. Nos outros casos é verdadeira.

Leis de De Morgan

¬(p ∧ q) ≡ ¬p ∨ ¬q e ¬(p ∨ q) ≡ ¬p ∧ ¬q. Estas equivalências são muito úteis!

Gerador de Tabelas-Verdade

Use: p, q, r, etc. para variáveis. Operadores: ¬ (negação), ∧ (e), ∨ (ou), → (condicional), ↔ (bicondicional)

Digite uma expressão lógica e clique em "Gerar" para ver a tabela-verdade.

Como usar

Sintaxe Básica

Use letras minúsculas para variáveis (p, q, r). Use parênteses para agrupar expressões.

Operadores Suportados

  • ¬ ou ! para negação
  • ∧ ou & para conjunção (E)
  • ∨ ou | para disjunção (OU)
  • → ou > para condicional
  • ↔ ou = para bicondicional

Exemplos

  • p ∧ q
  • ¬(p ∨ q)
  • (p → q) ↔ (¬q → ¬p)
  • (p ∧ q) ∨ (r ∧ ¬s)
